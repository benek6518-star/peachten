# Peach Ten

A lightweight mobile web puzzle game about clearing peach cards whose selected rectangle adds up to 10.

## How to Play

Open `index.html` in a web browser, then tap the start button. Drag across the board to select a rectangle. If the remaining active numbers inside the rectangle add up to exactly 10, those cards disappear.

## Run Locally

Double-click `index.html`, or open it from your browser with `File > Open`.

## Deployment

This game is ready for GitHub Pages as a static site. Upload these files together:

- `index.html`
- `style.css`
- `game.js`
- `README.md`

## Notes

- No ads
- No login
- No server
- No external SDKs
- Built with HTML, CSS, and JavaScript only
